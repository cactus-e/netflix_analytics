# netflix_analytics
# Аналитика Netflix Dataset

В ходе реализации проекта было проведено комплексное исследование продуктовых и финансовых метрик платформы Netflix: было развёрнуто локальное аналитическое хранилище данных (DWH) на базе PostgreSQL и реализован комплекс SQL-запросов напрямую внутри среды Jupyter Notebook.

## Стек технологий
* **Языки:** Python, PostgreSQL
* **Библиотеки:** Pandas · SQLAlchemy · Matplotlib · Seaborn
* **Среда:** Jupyter Notebook

## Ключевые результаты
1. **Смена географии:** Основной приток новых пользователей идет из регионов, следовательно стоит перераспределить рекламный бюджет для средних городов.
2. **Сезонный спад:** Каждый год в феврале активность пользователей резко падает, стоит перенести даты выхода новых сезонов сериалов на этот месяц.
3. **Технические неполадки:** Обнаружена проблема с форматом 4K. Стоит ускорить загрузку и уменьшить количество зависаний.
4. **Отток пользователей:** Стоит запустить рассылку с подборками новинок, чтобы не потерять аудиторию, которая не посмотрела ни минуты в течение определенного времени.

## Как запустить проект локально

Чтобы скачать все материалы и открыть проект в своём Jupyter Notebook, введите в терминале следующие команды:

```bash
# Клонируем проект на компьютер
git clone https://github.com/cactus-e/netflix_analytics.git

# Переходим в папку проекта
pip install pandas sqlalchemy matplotlib seaborn jupyter

# Запускаем Jupyter
jupyter notebook
```
